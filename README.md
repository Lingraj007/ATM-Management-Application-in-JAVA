# 🏧 ATM Management Application (Java)

A console-based ATM Management Application developed using **Java**, simulating real-world banking operations. The system allows users to perform transactions such as balance inquiry, cash withdrawal, deposit, and PIN authentication with secure validation.

## 🚀 Features

* User authentication with PIN verification
* Balance inquiry and mini statement
* Cash withdrawal with validation checks
* Deposit functionality
* Transaction handling with error messages

## 🛠️ Tech Stack

* **Language:** Java
* **Concepts Used:** OOP, Exception Handling, Collections

## 📌 Description

This project demonstrates core Java concepts and simulates basic ATM operations, focusing on user interaction, transaction management, and secure handling of account data. It is useful for understanding real-world banking workflows and backend logic.
